- Criar o projeto:
create-react-app themeswitcher --template=typescript

- Executar o projeto:
cd themeswitcher
yarn start

- Instalar o Styled Components
yarn add styled-components

- Instalar o react-switch
yarn add react-switch

- Instalar as definições do Styled Components para o Typescript:
yarn add @types/styled-components

- Instalar o Polished
yarn add polished
